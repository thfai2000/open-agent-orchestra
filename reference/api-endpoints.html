API Endpoints ​

All routes are served by OAO-API (Hono v4.6) at port 4002. OAO-UI proxies all /api/* requests to the API.

Authentication: JWT tokens signed with HS256, 7-day expiry, or Personal Access Tokens (PAT). Include in headers: Authorization: Bearer <token>

PATs use oao_ prefix (e.g., Authorization: Bearer oao_abc123...). PATs have fine-grained scopes controlling access.

Authentication ​

MethodPathAuthDescription
POST/api/auth/registerNoRegister new user (optional workspaceSlug, defaults to default)
POST/api/auth/loginNoLogin with email, password, optional provider (database/ldap). Returns JWT
GET/api/auth/meJWTCurrent user info + workspace
PUT/api/auth/change-passwordJWTChange password (database users only, LDAP users get 400)
GET/api/auth/providersNoList enabled auth providers for a workspace (query: ?workspace=slug)
