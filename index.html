AI Team with Segregation of Duties

Lower-cost agents handle screening and data gathering. Higher-cost agents with thinking capability solve complex problems. Just like a real team.

Zero Credential Exposure

Agents never see credentials. Scoped credentials are injected into MCP configs and HTTP headers via Jinja2 templates — secure by design.

🔄

Multi-Step Workflows

Chain agent steps with precedent output passing, variable injection, and per-step model/reasoning configuration.

⏰

Flexible Triggers

Cron schedules, exact datetimes, webhooks, system events with data matching, or manual execution.

🏢

Multi-Tenant Workspaces

Full workspace isolation with RBAC (super admin, workspace admin, creator, viewer). URL-scoped routing.

🧩

Plugin & MCP Ecosystem

Extend agents with Git-hosted plugins and Model Context Protocol (MCP) servers for custom tool integration.

Why Open Agent Orchestra? ​

Daily tasks can often be broken down into smaller subtasks that benefit from different AI capabilities at different cost levels. A lower-cost agent can screen and triage — then pass results to a higher-cost agent with specialized skills and reasoning power. OAO lets you build this kind of cost-effective AI team with proper credential management, audit trails, and segregation of duties.
